# ludo-dapp

## Instructions to install dependencies/run:

* Go to client/clientapp folder
  * npm init
  * npm i socket.io-client
  * npm i react-router-dom@5.2.0
  * npm install

* Go to server folder
  * npm init 
  * npm i express
  * npm i socket.io
  * npm i cors
  * npm install

# Run instructions:

* Go to client/clientapp/src folder
  * npm start
  
* Go to server folder
  * npm start
  
# open http://localhost:3000/ in browser
